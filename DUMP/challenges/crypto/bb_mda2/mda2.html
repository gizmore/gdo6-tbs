Do you think that this change made the algorithm any more secure.
Lets see how you can get on with the new one .

Maldito Algorithm :

Function Maldito ($text, $len)
{
      $diff = 10 ;
      $ct="";
      for ($x=0 ; $x < $len; $x++)
        {
        if ($x+$diff < $len-1)
                {
                $ct.=$text[$x] ^ $text[$x+$diff];
                }
        Else
                {
                $ct.=$text[$x] ^ $text[$diff];
                }
        if ((ord($text[$x]) % 2) == 0)
                {
                $diff++;
                }
        else
                {
                $diff--;
                }
        }
return $ct;
}

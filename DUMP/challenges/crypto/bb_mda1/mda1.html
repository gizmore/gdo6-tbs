The Maldito Algorithm (MDA 1).


Description:

This encryption method is based on some research work between a leading cryptographer and a rank amateur.
What the amateur came up was very surprising since even though the expert knew the plain texts and the cipher texts, and even had access to the algorithm - he was still unable to decrypt the simple example below. Even more surprising was that the only operation (The Maldito Operation) was a simple XOR operation. Lets see how you can get on with it.

Maldito Algorithm :

Function Maldito ($text, $len)
{
      $diff = 10 ;
      $ct="";
      for ($x=0 ; $x < $len; $x++)
        {
        if ($x+$diff < $len-1)
                {
                $ct.=$text[$x] ^ $text[$x+$diff];
                }
        Else
                {
                $ct.=$text[$x] ^ $text[$diff];
                }
        if ((ord($ct[$x]) % 2) == 0)
                {
                $diff++;
                }
        else
                {
                $diff--;
                }
        }
return $ct;
}
